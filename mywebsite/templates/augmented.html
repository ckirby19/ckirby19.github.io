Augmented Reality Headset

  • Company/Group: Bose
  • My Role: Mechanical Engineer
  • Dates: June 2018 - December 2018

The Bose Prototyping group were interested in designing a augmented reality headset for use with Bose Spatial Audio Technology. The design would include head and hand tracking, based on an open source concept by Leap Motion. This was an open-ended project, so the requirements formed over time.

  • Integrated and accurate hand tracking to create a seamless experience for the user
  • Highest resolution screens possible for the given size restrictions around the persons head
  • 50/50 transmission-reflection ratio for the optics to create a visually attractive overlay for the user
  • Provide a comfortable fit to the user for extended use
  • Head tracking to create an experience that can be linked to the physical room
  • Spatial audio experience to showcase the use case of this technology with immersive audio

As an example of the iteration process, the optics went through a number of versions to obtain the targeted 50/50 transmission-reflection ratio so that the user can see the outside world clearly as well as the reflection from the screens. In the first attempt, clear acrylic was machined into the desired shape and mirror film was then appled to the outer face. However, the flat mirror film did not cleanly form to the curved optics, despite attempted applications with window spray and heat. Next, using the same clear acrylic, we attempted to get the correct ratio by using a thin layer of mirror spray. This attempt is shown in "Version 1 of hand tracking", and it clearly did not give the right ratio. The most recent version of the optics were made by finding acrylic with transmission-reflection ratio as close as possible to the desired one, machining a block for the acrylic to be thermoformed onto and then cutting the acrylic into the desired shape. The optics still require improvements to increase optical clarity and reduce imperfections.

Finally, I Integrated Bose Spatial Audio technology into the frame, and created a simple spatial audio experience on Unity, where a user can move a set of objects around a room to change the direction from which the audio is being emitted. This device was used to display the potential for augmented reality to create new audio experiences for both customers and researchers.
